# My Portfolio Website - Overview

This repository contains the open source version of my portfolio website.

## Notes

This project uses the standard `gsap` package, including `ScrollSmoother` and `SplitText`.
Check the current GSAP installation and license terms before redistributing or reusing the animation setup:
https://gsap.com/docs/v3/Installation/

Client names, product names, screenshots, logos, and other third-party marks shown in the portfolio belong to their respective owners and are included only to identify past work.

**Tech stack** - React, TypeScript, GSAP, ThreeJS, WebGL, HTML, CSS, JavaScript
